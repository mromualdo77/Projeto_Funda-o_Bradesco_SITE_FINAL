Tecnologia - Realidade Aumentada

De uma forma simples, Realidade Aumentada é uma tecnologia que permite que o mundo virtual seja misturado ao real, possibilitando maior interação e abrindo uma nova dmensão na maneira como nós executamos tarefas, ou mesmos as que nós incubimos às máquinas. Assim, se você pensava que objetos pulando para fora da tela era elementos de filmes de ficção cientifica, está na hora de mudar seus conceitos. Aliás, o que acontece com a Realidade Aumentada é o contrário: você pualra para dentro do mundo virtual para interagir com objetos que so estaão limitados à sua imaginação. - tecmundo.com.br/realidade-aumentada - acesso 11/2021

Acessibilidade na web significa que as pessoas com deficiência podem usar a web.
